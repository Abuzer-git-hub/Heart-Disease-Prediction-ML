# Heart Disease Prediction using Machine Learning

## Overview
This project predicts the presence of heart disease using multiple machine learning classification models. It uses the Cleveland dataset for training and independent datasets such as Hungarian, Switzerland, and VA for testing.

## Objectives
- Predict heart disease using patient medical records
- Compare multiple machine learning models
- Perform preprocessing and feature engineering
- Handle imbalanced data using SMOTE
- Evaluate model performance using classification metrics

## Dataset
The project uses UCI Heart Disease datasets. Features include age, chest pain type, cholesterol, blood pressure, heart rate, ECG results, and other medical indicators.

## Machine Learning Models Used
- Logistic Regression
- K-Nearest Neighbors
- Decision Tree
- Naive Bayes
- Support Vector Machine
- Random Forest

## Evaluation Metrics
- Accuracy
- Precision
- Recall
- F1-Score
- Confusion Matrix
- MSE
- ROC-AUC Score

## Expected Output
- Heart disease prediction
- Model accuracy comparison
- Confusion matrices
- Performance graphs
- Best model identification

## Future Improvements
- Deploy as a web application
- Use larger real-world hospital datasets
- Apply deep learning techniques
- Add real-time healthcare monitoring

## Conclusion
This project demonstrates how machine learning can support healthcare analytics by predicting heart disease risk using medical features.
